# Chem Games (Vite + React Router)

This app uses React Router with routes:

- `/`
- `/login`
- `/signup`
- `/discover`
- `/saved`
- `/profile`

## Environment variables

Set these in `.env`:

```env
VITE_SUPABASE_URL=your-project-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```

## Database setup

Run the SQL in `SUPABASE_SETUP.md` inside your Supabase SQL Editor.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).
